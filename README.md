 Mi Portfolio Personal - Preentrega 2 

Bienvenido al repositorio de mi segundo proyecto para el curso de Desarrollo Web en Coderhouse. En esta etapa, el enfoque estuvo en la organización de archivos, optimización de recursos y diseño responsive.

 Sobre mí
Mi nombre es Agustín Szulman y este sitio es un reflejo de mi trayectoria y mis pasiones.

 Tecnologías utilizadas
HTML5: Estructura semántica.
CSS3: Estilos avanzados, Flexbox y CSS Grid.
GitHub Pages: Despliegue y hosting de la web.

Características del Proyecto
Diseño Responsive: Adaptado para Desktop, Tablet y Mobile mediante Media Queries.
Arquitectura de Archivos: Estructura prolija dividida en carpetas (`css/`, `img/`, `pages/`).
Navegación Fluida: Enlaces funcionales entre todas las secciones del sitio.
